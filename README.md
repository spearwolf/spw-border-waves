# spw-border-waves

## Usage

1. Build

```sh
$ npm run build
$ copy build/spw-border-waves.js $TO_YOUR_PUBLIC_STATIC_ASSETS_DIRECTORY
```

2. Integrate as web component

```html
<script type="module" src="spw-border-waves.js"></script>

<spw-border-waves color="#666"></spw-border-waves>
```

## Development

```sh
$ npm run start
```
